# iCOBRA IDE — Keyboard Shortcuts Reference

All shortcuts use a Bluetooth keyboard. On devices without a hardware keyboard, use the Command Palette (≡ button or Ctrl+Shift+P).

## General

| Action | Windows/Linux | Mac |
|--------|---------------|-----|
| Command Palette | Ctrl+Shift+P | ⌘+Shift+P |
| Quick Open (Go to File) | Ctrl+P | ⌘+P |
| New Window | Ctrl+Shift+N | ⌘+Shift+N |
| Close Editor | Ctrl+W | ⌘+W |
| Settings | Ctrl+, | ⌘+, |
| Keyboard Shortcuts | Ctrl+K Ctrl+S | ⌘+K ⌘+S |
| Zoom In | Ctrl+= | ⌘+= |
| Zoom Out | Ctrl+- | ⌘+- |
| Reset Zoom | Ctrl+0 | ⌘+0 |
| Toggle Fullscreen | F11 | Ctrl+⌘+F |
